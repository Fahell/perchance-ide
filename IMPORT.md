# Import URL

## Option 1: HTML Panel (standalone generator)

Add an HTML panel to your generator with:

```html
<script>
  import("https://cdn.jsdelivr.net/gh/Fahell/perchance-ide@c30c446/dist/agent.js");
</script>
```

Also add `agentAi = {import:ai-text-plugin}` to your list panel.

## Option 2: Perchance Custom Code (legacy)

```js
import("https://cdn.jsdelivr.net/gh/Fahell/perchance-ide@c30c446/dist/agent.js");
```
